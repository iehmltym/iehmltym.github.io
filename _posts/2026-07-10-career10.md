## 3. 代码示例：RAG Search

```python
@router.get("/api/v1/rag/search")
async def rag_search(query: str):
    docs = retriever.invoke(query)

    return {
        "query": query,
        "selected_tool": "manual_search",
        "retrieved_count": len(docs),
        "results": docs
    }
```

### 面试说法

```text
問い合わせ内容をqueryとして受け取り、Retrieverを使って関連ドキュメントを検索しました。
検索対象は、製品マニュアル、保守FAQ、問い合わせ履歴です。
```

## 4. 代码示例：Tool Calling

```python
def route_tool(query: str) -> str:
    if "操作" in query or "設定" in query:
        return "manual_search"

    if "エラー" in query or "故障" in query:
        return "faq_search"

    return "history_search"
```

### 面试说法

```text
問い合わせ内容に応じて検索対象を切り替える簡易的なTool Calling処理を実装しました。
```
